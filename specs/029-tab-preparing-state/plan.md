# Implementation Plan: Show Preparing Dashboard Component Based on a Condition

**Branch**: `029-tab-preparing-state` | **Date**: 2026-05-01 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `/specs/029-tab-preparing-state/spec.md`

## Summary

Extend `useDashboardStatusPolling` with three per-tab "stale" flags (`isRecommendationTabStale`, `isWorkforceTabStale`, `isIndustryTabStale`). A flag is `true` when the corresponding tab's status is `pending` **and** its `updatedAt` timestamp is older than 5 minutes. Thread these flags as an `isStale` prop into each tab page component. When `isStale` is `true`, the tab page renders only `<PreparingDashboard />` instead of its normal children.
