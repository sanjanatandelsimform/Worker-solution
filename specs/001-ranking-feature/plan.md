## Summary

Replace the static placeholder label in Question 2 of the Goals section in `AdditionalQuestions.tsx` with the existing `RankingList` drag-and-drop component. The ranking list is fed from the user's checked goals (Question 1), enforces a minimum of 3 selected goals before submission, and produces a dynamic `workforceGoalsRanking` array in the Finch assessment payload — replacing the hardcoded static constant that always sent the same 3 values.

**Core approach**: Mirror the implementation in `DynamicQuestionRenderer.tsx` (line 1142) — import `RankingList`, derive `availableOptions` from `goalsData` filtered by `selectedGoals`, wire `onChange` to `topThreeGoals` state, and validate minimum 3 before submit.
